# ms (MicroService)

ms (MicroService) — Go library with microservice related solutions.

## Packages

* [`ms`](./ms) — core microservice primitives (HTTP path normalizer, types)
* [`ms/msconfig`](./ms/msconfig) — HTTP settings, common headers, round-trip wrap
* [`msfx`](./msfx) — [uber-go/fx](https://github.com/uber-go/fx) module wiring
